# Uncharted Terra 2264
A Nintendo 64 first-person puzzle adventure throught a colony on another planet made with Libdragon and Tiny3D for the N64Brew GameJam #6.

# Description:

```
There is a remote exoplanet colony that was jointly inhabited by humans and an alien species known only by fragments of their designation: The Irenix.
The colony was abandoned generations ago after a cataclysmic failure of a shared project - an energy-manipulating device deep inside the settlement.
Rumors suggest that restoring this device could reveal what happened to the colonyâ€™s inhabitants and discover a limitless power source, but the site is considered too unstable to explore.

No one outside of the colony really knows what the artifact really was, or how it failed, or what happened to the colonists. 
There were attempts at exploration beforehand, but no one came back. 

That does not stop you anyway.
```

This game uses advanced features, not previously seen in any N64 official title, that is a result of extensive research and optimization.
Expansion Pak is ___NOT___ needed, but recommended to play this game.

There's a walkthrough of the game in the walkthrough.txt file

# Known issues:
- EEPROM may not work on older Everdrive flashcart models, or with outdated software (libdragon incompatibility)
- Beating the game and starting a new game may cause a crash without expansion pak.

# Controls:
- Stick: look around
- D-Pad/C-Pad: move around
- Z trigger: crouch
- L/R triggers: sprint


# Features:
- Multilanguage support with 8 different languages using .ini configs and [tortellini](https://github.com/Qix-/tortellini) open-source library
- Stereo music, FMV cutscenes and stereo audio on a 8MB cartridge
- 360i TrueColor 3d graphics ___without the expansion pak___, extremely rare in N64 titles
- 3D graphics with antialiasing and high-res fullcolor mipmapped textures
- Dynamic hardware bumpmapping on most surfaces
- HDR rendering and bright lighting with autoexposure
- Simple bloom implementation
- Rumble Pak and EEPROM save support
- Save the game anytime during gameplay, HL1 style
- Basic N64 mouse support


# Build the game
- Install [Libdragon SDK](https://github.com/SpookyIluha/libdragon/tree/unstable-modified) (unstable-modified)
- Install [Tiny3D](https://github.com/HailToDodongo/tiny3d) library
- Clone this repository
- Build it with make command
